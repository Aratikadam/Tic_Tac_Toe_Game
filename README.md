# Tic Tac Toe Game

A simple Python console game for two players to play Tic Tac Toe. Take turns placing X and O on a 3×3 grid. The first player to align three marks vertically, horizontally, or diagonally wins!

## Features

* Two-player gameplay (X and O)
* Detects wins, draws, and invalid moves
* Simple console interface
* Shows current board after each move
* Easy to understand and play

## Requirements

Python 3.x (no additional packages needed)

## Usage

1. Clone the repository:

```bash
git clone https://github.com/Aratikadam/Tic_Tac_Toe_Game.git
```

2. Run the game:

```bash
python main.py
```

3. Follow the on-screen instructions to enter your move (row and column).
